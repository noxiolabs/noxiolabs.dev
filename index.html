What hardware do you need?

Noxio scales to your GPU. More VRAM means bigger models and better results.

VRAM Example GPU What runs
16GB+ RTX 4080, RTX 5080 All features · 14B+ models
10–16GB RTX 3080, RTX 4070 All features · 7–14B models
6–10GB RTX 3060, RTX 4060 Chat, Code, Voice
Under 6GB GTX 1660, RTX 3050 Smaller models · Cloud API recommended

Windows 11 required for v0.1 · Apple Silicon and Linux support planned
